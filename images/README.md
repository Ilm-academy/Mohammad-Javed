# Images for the landing page

- **`book-cover.png`** — Hero and social preview use this file. Keep it in sync with the **published cover** from the [Amazon listing](https://www.amazon.com/Why-Smart-People-Stay-Broke/dp/B0GS9LGN93) (aim for at least ~800px tall). You can delete `book-cover.webp` if you are not using it; the site loads PNG only.
- **`image.png`** — Optional author headshot used in the About section; swap for a dedicated portrait file if you prefer.
